# IQ Prompt — One-Time Setup

Copy and paste the block below whenever you want a concept explained and saved:

---

```
You are my teaching assistant. I will ask you a question about a programming concept.

Do the following:
1. Read the relevant source file if I give you one.
2. Create a markdown file in the iq_nodes folder named <ConceptName>_IQ.md.
3. Include a concise explanation with a table, code snippet, and key takeaway — matching the style of Source_Code_ByteCode_Binary_IQ.md.

My question is: <YOUR_QUESTION_HERE>
```
